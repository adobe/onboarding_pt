# 透明効果のサポート
プロジェクトで不透明度などの透明効果プロパティを使用します。**ファイル／サンプルを開く／フレンチレストランテーブル**に進んで、新しいサンプルプロジェクトでこれらの機能を確認できます。

# パスの改善
「パスに沿ってペイント」では、**カスタム接線**を有効にして頂点の接線を編集できるようになりました。また**パスリスト**で「プロパティ」を右クリックして、あるパスから別のパスにプロパティをコピー＆ペーストできるようになりました。
